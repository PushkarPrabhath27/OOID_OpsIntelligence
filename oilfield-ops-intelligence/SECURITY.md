# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our intelligence platform seriously. If you find a vulnerability, please report it via the following process:

1. Please do not open a public GitHub issue.
2. Email the security team directly at `security@ooid-project.org`.
3. You will receive a response within 48 hours.
4. We will coordinate a fix and public disclosure.
